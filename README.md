## Description
In this project i have developed Postman collection to test all CRUD operations using various features of postman like:
1) test/pre-request scripts.
2) validation using Postman's Javascript based PM library.
3) Data driven testing using JSON and CSV.
4) Dynamic data driven testing using dynamic variables.
5) Different variables , Global , environment , collection and local variables.
6) Request chaining.
7) Execution of postman collection using Newman CLI.
8) Auto retry of APIs for fixed number of iterations.
9) Reports generation using html , htmlextra and newman CLI.
10) Authentication and authorization.
